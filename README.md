# Projetos de Engenharia de Dados

Repositório contendo projetos práticos de Engenharia de Dados com foco em ETL/ELT, orquestração, APIs, bancos de dados e pipelines automatizados.

## 

### project_etl_weather
- Pipeline ETL automatizado para coleta e processamento de dados climáticos via API REST.
- **Tecnologias:** Python, Apache Airflow, Docker, PostgreSQL, Pandas, SQLAlchemy, Parquet, OpenWeather API, Jupyter Notebook.


### project_dataform_dw
- Pipeline de dados seguindo arquitetura medalhão (bronze/silver/gold) com modelagem dimensional (star schema), usando dados públicos de e-commerce.
- **Tecnologias:** Dataform, BigQuery, SQL, SQLX, Google Cloud Platform (GCP).
